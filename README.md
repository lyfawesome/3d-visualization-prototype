# 3D Visualization Prototype

This repository contains the planning materials for a Windows desktop prototype that demonstrates 3D specimen model visualization for a control and drive system response.

## Goal

Build a demonstrable 3D visualization prototype that can:

- Load specimen model files generated by professional 3D/CAD software.
- Automatically create an interactive 3D visualization scene.
- Mark and display sensor positions on the specimen model.
- Map simulated load data to the model surface as a color contour visualization.

The current repository is for planning and handoff only. It does not yet contain implementation source code.

## Planned Technical Stack

- Qt for the Windows desktop application.
- VTK for 3D rendering, interaction, labels, color mapping, and screenshot export.
- OCCT for reading STEP/STP CAD model files.

Priority model formats:

- STEP/STP
- STL
- OBJ

The first stage focuses on a credible demonstration prototype for response-file evidence. Real DAQ/PLC integration, engineering-grade simulation validation, and full product hardening are planned as later extensions.
